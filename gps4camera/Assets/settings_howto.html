How to

For using this app to record tracks:

Press the record button to enable GPS recording. Press the stop button to stop recording.

At any time, the recorded tracks can be exported from the Files tab. Either by swipping left and tapping Share or tap the file and then tap on Share. Files can be shared using any sharing method available to your device such as email or text message.

For using the app to record tracks for geo tagging

Follow the steps above to record and export tracks.

At some point during GPS track recording, before, or after, tap on Sync. Then use the camera you plan to geo tag photos from to take a picture of the Sync tab. It's recommended to select a shutter speed quick enough to keep the QR code sharp, otherwise the QR decoding step will fail.

Collect the images and track file into the same directory on a Mac (PC support not yet available).

Exiftool MUST be installed before running qrfinder. To install exiftool on Mac.

Run qrfinder "[path to images]"

If all goes well, qrfinder will locate the QR coded image, extract the date, then using exiftool, apply the date correct to all images in the directory. Then it will use exiftool to geo tag the images in the directory, as long as a .gps or .kml file can be found.
